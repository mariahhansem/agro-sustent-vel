# agro-sustent-vel
É um site desenvolvido para apresentar a importância da agricultura moderna aliada à sustentabilidade.
Ele tem como objetivo mostrar que é possível aumentar a produção agrícola utilizando tecnologia e boas práticas sem prejudicar o meio ambiente. O site está organizado em seções que explicam a relação entre produção sustentável, preservação ambiental e o papel da inovação no campo.
Com um design inspirado na natureza, predominando tons de verde e elementos visuais ligados ao meio rural, o projeto transmite a ideia de equilíbrio e responsabilidade ambiental. Além disso, conta com interatividade básica em JavaScript, tornando a experiência do usuário mais dinâmica.
Em resumo, o site reforça a mensagem de que o futuro do agronegócio depende da união entre eficiência produtiva e preservação dos recursos naturais, promovendo um modelo de desenvolvimento sustentável.
